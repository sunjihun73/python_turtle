# Python Study Project - Python Turtle & 기초 문법 연습

이 프로젝트는 Python의 `turtle` 그래픽 모듈과 기본적인 Python 문법을 학습하기 위한 예제 코드 모음입니다.

## 📁 주요 파일 설명

### 🐢 Turtle Graphics 예제
- **main.py**: 두 마리의 거북이가 화면을 이동하는 기본적인 turtle 사용법 예제입니다.
- **turtle2.py**: 마우스 클릭 이벤트를 활용하여 화면에 그림을 그리는 예제입니다. (왼쪽 클릭: 그리기, 오른쪽 클릭: 이동, 휠 클릭: 거북이 크기 및 색상 변경)
- **turtle3.py**: 거북이가 지정된 영역 안에서 무작위로 움직이며 선을 그리는 예제입니다. 영역을 벗어나면 중앙으로 돌아오며, 5번 반복 후 종료됩니다.
- **chapter5.py**: `for` 반복문을 활용하여 무지개색의 다양한 크기의 원을 그리는 예제입니다.

### 📚 Python 기초 문법 연습
- **chapter3.py**: 문자열 포맷팅(`%`, `.format()`)과 중첩된 반복문을 이용한 별 찍기 예제입니다.
- **chapter3-2.py**: 변수 선언, 자료형 확인, 산술 연산자, 조건문 등의 기초 문법 실습 코드입니다.

## 🚀 실행 방법

각 파이썬 파일을 직접 실행하여 결과를 확인할 수 있습니다.

```bash
python main.py
python turtle2.py
# 또는 다른 파일들 실행
```

*참고: `turtle` 모듈을 사용하는 코드는 GUI 환경에서 실행해야 창이 정상적으로 나타납니다.*
